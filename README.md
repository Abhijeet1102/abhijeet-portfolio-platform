# Abhijeet Rai Portfolio Platform

This is an enterprise-grade monorepo containing the frontend, backend, and documentation for the Abhijeet Rai Portfolio Platform.

## Apps

- `web`: Next.js App Router frontend.
- `server`: Express backend API.
- `docs`: Next.js MDX documentation site.

## Packages

- `ui`: Shared UI components using Tailwind CSS and Shadcn UI.
- `types`: Shared TypeScript interfaces.
- `config`: Shared constants and environment configs.
- `eslint-config`: Shared ESLint configurations.
- `typescript-config`: Shared `tsconfig.json` bases.

## Getting Started

1. Install dependencies: `pnpm install`
2. Run development servers: `pnpm turbo run dev`
